# Wedding-as-a-Service Platform

A modern, high-performance wedding platform built to handle RSVPs, real-time analytics, and secure digital gifting. 

## Tech Stack
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS + Framer Motion
* **Database & Real-time:** Supabase
* **Payments:** Safaricom Daraja API (M-Pesa STK Push)
* **Deployment:** Vercel

## Features
* Interactive, animated UI with scroll reveals and soundscapes.
* Secure RSVP gateway with unique guest access codes.
* Real-time Admin Dashboard for headcount and gift tracking.
* Native M-Pesa integration for seamless registry contributions.

## Local Development
1. Clone the repository.
2. Run `npm install`.
3. Set up your `.env.local` with Supabase and Daraja credentials.
4. Run `npm run dev` to start the local server.
